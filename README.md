# Live Stocks App - React

### Tech Stack:

1) React.js
2) Redux
3) Bootstrap 4 CSS 
4) Moment.js 

### To install:

1) git clone `https://github.com/agulhane/live-stocks-app.git`
2) cd `live-stocks-app`
3) install the dependencies using `npm install` (takes 2-3 minutes)
4) To start the app, use `npm start`
